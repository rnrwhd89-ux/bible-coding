# 성경 앱 QA 테스트 가이드

## 테스트 환경 설정
- URL: https://bible-app-29a53.web.app
- 브라우저: Chrome, Safari, Firefox 모두 테스트
- 화면 크기: 모바일(375x667), 태블릿(768x1024), 데스크톱(1920x1080)
- 개발자 도구 열고 콘솔 에러 확인하면서 테스트

## 1. 기본 성경 읽기 기능

### 1.1 페이지 네비게이션
- [ ] 좌측 상단 "창세기 1장" 드롭다운 클릭
- [ ] 책 목록이 표시되는지 확인
- [ ] 다른 책(예: 요한복음) 선택 시 이동하는지 확인
- [ ] 장 번호 클릭하여 다른 장으로 이동
- [ ] 이전/다음 버튼으로 장 이동 테스트

### 1.2 번역본 선택
- [ ] 우측 상단 "개역개정" 버튼 클릭
- [ ] 번역본 목록(개역개정, 개역한글, NIV) 표시 확인
- [ ] 각 번역본 선택 시 구절이 바뀌는지 확인
- [ ] **중요**: 개역개정은 Firestore에서 데이터를 가져옴 (인덱스 생성 완료 후 작동)
- [ ] 개역한글, NIV는 Bolls API에서 데이터를 가져옴

## 2. 구절 선택 기능 (중요!)

### 2.1 단일 선택
- [ ] 구절 1번 클릭
- [ ] 노란색 박스로 하이라이트 표시 확인
- [ ] 하단에 선택 도구 패널 표시 확인
- [ ] "빌립보서 3:1 (1절 선택)" 형식으로 표시 확인

### 2.2 복수 선택 (Cmd+Click / Ctrl+Click)
- [ ] 구절 1번 클릭 (선택)
- [ ] **Cmd+Click**(Mac) 또는 **Ctrl+Click**(Windows)로 구절 3번 클릭
- [ ] 구절 1, 3이 동시에 선택되는지 확인
- [ ] "빌립보서 3:1,3 (2절 선택)" 표시 확인
- [ ] 선택된 구절 다시 Cmd+Click 시 선택 해제 확인

### 2.3 범위 선택 (Shift+Click) - 신규 수정!
- [ ] 구절 1번 클릭 (선택)
- [ ] **Shift+Click**으로 구절 5번 클릭
- [ ] 구절 1, 2, 3, 4, 5 모두 선택되는지 확인
- [ ] "빌립보서 3:1-5 (5절 선택)" 표시 확인
- [ ] **기대 결과**: 중간 구절(2, 3, 4)도 모두 선택되어야 함

### 2.4 드래그 범위 선택 (선택 테스트)
- [ ] 구절 1번에서 마우스 다운
- [ ] 구절 5번까지 드래그
- [ ] 마우스 업
- [ ] 구절 1~5 범위 선택 확인
- [ ] 모바일(터치): 구절 1에서 터치 시작 → 5까지 스와이프 → 범위 선택 확인

### 2.5 선택 취소
- [ ] 구절 여러 개 선택
- [ ] 선택 도구 패널 우측 상단 X 버튼 클릭
- [ ] 모든 선택 해제 확인
- [ ] 선택 도구 패널 사라짐 확인

## 3. 하이라이트 기능

### 3.1 색상 하이라이트
- [ ] 구절 1~3 범위 선택
- [ ] 선택 도구 패널에서 5가지 색상 버튼 각각 클릭
  - 노란색, 초록색, 파란색, 분홍색, 보라색
- [ ] 선택한 구절의 배경색이 즉시 변경되는지 확인
- [ ] 좌측에 색상 테두리가 표시되는지 확인

### 3.2 하이라이트 변경
- [ ] 이미 하이라이트된 구절 선택
- [ ] 다른 색상 클릭
- [ ] 색상이 변경되는지 확인

### 3.3 하이라이트 제거
- [ ] 하이라이트된 구절 선택
- [ ] 동일한 색상 버튼 다시 클릭 (토글)
- [ ] 하이라이트가 제거되는지 확인

## 4. 메모 기능 (수정 완료!)

### 4.1 메모 작성
- [ ] 구절 1~3 선택
- [ ] "새 메모" 버튼 클릭
- [ ] 메모 모달 창이 열리는지 확인
- [ ] 자동 제목: "[책이름] [장:절] 메모" 형식 확인
- [ ] 제목 입력: "하나님의 은혜"
- [ ] 내용 입력: "이 말씀은 하나님의 은혜와 자비에 대해 깊은 의미를 담고 있습니다."
- [ ] "저장" 버튼 클릭
- [ ] 모달 닫힘 확인
- [ ] 구절 옆에 **노란 점** 생성 확인

### 4.2 메모 목록 조회 - 수정 완료!
- [ ] 하단 네비게이션 바에서 "메모" 탭 클릭
- [ ] 저장한 메모가 목록에 표시되는지 확인
- [ ] 메모 제목, 구절 참조, 내용 미리보기 표시 확인
- [ ] "N개의 메모" 카운트 표시 확인
- [ ] **기대 결과**: "아직 메모가 없습니다" 메시지가 아니라 실제 메모 목록이 표시되어야 함

### 4.3 메모 상세보기 - 수정 완료!
- [ ] 방법 1: 메모 탭에서 메모 항목 클릭
- [ ] 방법 2: 성경 본문에서 **노란 점** 클릭
- [ ] 메모 상세보기 페이지 열림 확인
- [ ] 구절 참조(예: "빌립보서 3:1-3") 표시 확인
- [ ] 메모 제목 표시 확인
- [ ] 메모 내용 전체 표시 확인
- [ ] **기대 결과**: 빈 페이지가 아니라 메모 내용이 정상 표시되어야 함

### 4.4 메모 수정
- [ ] 메모 상세보기 페이지에서 "편집" 버튼 클릭
- [ ] 제목 수정: "은혜와 자비"
- [ ] 내용 수정: 텍스트 추가/수정
- [ ] "저장" 버튼 클릭
- [ ] 수정 내용 반영 확인

### 4.5 구절 참조 클릭
- [ ] 메모 상세보기 페이지에서 구절 참조(예: "빌립보서 3:1-3") 클릭
- [ ] 성경 본문 탭으로 이동하는지 확인
- [ ] 해당 책의 해당 장이 표시되는지 확인
