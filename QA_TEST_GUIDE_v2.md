## 🔴 최우선 테스트: BUG #5 수정 확인

### TEST #1: 구절 클릭 기본 동작
**목표:** 구절 클릭 시 흰 페이지로 이동하지 않는지 확인

**테스트 순서:**
1. [ ] 번역본을 **개역한글**로 설정
2. [ ] **빌립보서 3장** 선택
3. [ ] **구절 1 텍스트 영역** 클릭
4. [ ] ✅ **예상 결과:** 구절이 선택되고 하단에 메뉴(하이라이트/메모/AI질문) 표시
5. [ ] ❌ **실패 조건:** 흰 페이지로 이동하거나 페이지가 새로고침됨

**추가 테스트:**
- [ ] 구절 2, 3, 4도 각각 클릭해서 동일하게 작동하는지 확인
- [ ] 번역본을 **NIV**로 변경 후 동일 테스트 반복
- [ ] 다른 책(**마태복음 1장**)에서도 테스트

**결과 기록:**
```
✅ 성공 / ❌ 실패
문제 발견 시 상세 내용:
```
