# 📖 바이브 성경 앱

Firebase 기반 성경 읽기 및 메모 앱

## ✨ 주요 기능

- 📚 **성경 읽기**: 개역한글, NIV 번역본 지원
- 🖍️ **형광펜**: 5가지 색상으로 중요 구절 표시
- 📝 **메모**: 구절별 메모 작성 및 편집
- 🤖 **AI 채팅**: Groq API를 활용한 성경 질문답변
- ✅ **읽기표**: 성경 진도 체크
- 🔄 **Firebase 동기화**: 모든 기기에서 데이터 동기화

## 🚀 시작하기

### 1. 저장소 클론
```bash
git clone <repository-url>
cd bible-app
```

### 2. 패키지 설치
```bash
npm install
```

### 3. 개발 서버 실행
```bash
npm start
```

브라우저에서 http://localhost:3000 접속

## 📦 배포

```bash
npm run deploy
```

## 🔧 환경 설정

Firebase 프로젝트 설정은 `src/firebase.js`에 있습니다.
